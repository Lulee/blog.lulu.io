# 提高工作效率的工具

  1. 截图贴图工具snipaste

    snipaste不仅有截图功能，还有贴图功能（截图能「钉」在当前屏幕上)，还有取色功能。在开发过程中需要对着原型开发时，有了这个神器，就可以免除来回切换窗口了，还便于取色，方便高保真还原设计稿。

    下载地址：https://www.snipaste.com/

    常用快捷键 描述
    f1 截图
    f3 贴图
    ctrl + c 保存到剪贴板
    ctrl + T 保存到贴图
    esc 取消截图
    ，或. 回放截图记录
    alt 放大镜
    ⌥ + 鼠标 + c 在放大镜的情况下，移动鼠标，选取想要取色的点，然后按 c 复制色值
